# 🌍 Traveloop

AI-powered travel planner that creates dynamic itineraries using real-time data.

## 🚀 Features
- 🔐 User Authentication
- 📍 City-based trip planning
- 🤖 AI-generated itineraries
- 📅 Day-wise planning
- ☁️ Cloud deployment (Render)

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: MongoDB
- Deployment: Render

## ⚙️ Setup

### Clone repo
```bash
git clone https://github.com/YOUR_USERNAME/traveloop.git
cd traveloop
```

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

---

## 🌐 Deployment
- Frontend: Render Static Site
- Backend: Render Web Service
- Database: MongoDB Atlas
